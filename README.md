# MediaVault

Self-hosted домашний архив файлов/медиа: загрузка через браузер, поиск, альбомы, дубликаты, можно пользоваться вместе с помощью ссылки.

## Что такое Self-hosting?

Self-hosting (селф-хостинг) - это когда Вы запускаете сервис на своём железе или своём сервере, а не пользуетесь чужим готовым облаком.

## Зачем использовать именно это решение?

Несмотря на то что сегодня существует много облачных сервисов для хранения медиа, многие предпочитают self-hosted решения. Они позволяют хранить файлы на собственной инфраструктуре и не зависеть от сторонних платформ.
MediaVault как раз решает эту задачу: он помогает организовать личную медиатеку, управлять файлами и получать удобный доступ к коллекции из одного места.
Это особенно полезно, когда у Вас уже есть NAS, домашний сервер или просто большая библиотека фото, видео и других медиафайлов, которые хочется держать полностью под собственным контролем.

## Быстрый старт (Docker)

1. Создайте `.env` из примера:

   - [`.env.example`](file:///workspace/.env.example)

2. Запустите:

```bash
docker compose up --build
```

Откройте: http://localhost:8080

## Первый админ

Админ создаётся на старте через env:

- `BOOTSTRAP_ADMIN_EMAIL`
- `BOOTSTRAP_ADMIN_PASSWORD`

В Docker это можно добавить в `docker-compose.yml` (environment).
